Кафе «Aroma» — демо-сайт

Как опубликовать онлайн быстро:
1) Netlify → Add new site → Deploy manually → перетащи папку cafe
2) GitHub Pages → включи Pages из ветки main
3) ISPmanager → загрузить файлы в папку домена и включить SSL

Где менять цены и блоки:
- Цены: секция #pricing внутри index.html
- Услуги/описания: секция #services
- Галерея: секция #work (.thumb можно заменить на реальные изображения)
